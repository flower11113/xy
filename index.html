Kvadratni Korijen

Kvadratni korijen broja x je broj koji, kad se pomnoži sam sa sobom, daje x.

Primjer: √9 = 3, jer 3 * 3 = 9.

Kubni Korijen

Kubni korijen broja x je broj koji, kad se pomnoži sam sa sobom tri puta, daje x.

Primjer: ∛8 = 2, jer 2 * 2 * 2 = 8.

Aritmetički Korijen

Aritmetički korijen broja x je broj koji, kad se pomnoži sam sa sobom n puta, daje x.

Primjer: √25 = 5, jer 5 * 5 = 25.
